Tailwind CSS Tutorial

Introduction to Tailwind CSS

Tailwind CSS is a utility-first CSS framework that allows you to build custom designs without having to write a lot of custom CSS. It provides low-level utility classes to build complex designs.

Utility Classes

Utility classes are single-purpose classes that you can use to style individual elements.

<div class="bg-blue-500 text-white p-4">This is a blue box</div>
<div class="text-center text-gray-700">Centered text</div>

Responsive Design

Tailwind provides utility classes to create responsive designs easily.

<div class="md:flex">
  <div class="md:w-1/2 p-4 bg-green-200">Left Column</div>
  <div class="md:w-1/2 p-4 bg-green-400">Right Column</div>
  </div>

Flexbox

Tailwind includes a comprehensive set of flexbox utilities.

<div class="flex">
  <div class="flex-1 p-4 bg-red-200">Flex Item 1</div>
  <div class="flex-1 p-4 bg-red-400">Flex Item 2</div>
  </div>

Grid System

Tailwind's grid system is based on CSS Grid Layout.

<div class="grid grid-cols-3 gap-4">
  <div class="bg-blue-200 p-4">Grid Item 1</div>
  <div class="bg-blue-400 p-4">Grid Item 2</div>
  <div class="bg-blue-600 p-4">Grid Item 3</div>
</div>

Typography

Tailwind offers a variety of typography utilities to style text.

<h1 class="text-4xl font-bold text-blue-500">Heading 1</h1>
<p class="text-lg text-gray-700">This is a paragraph with large text.</p>
<a href="#" class="text-green-500 underline">This is a link</a>

Spacing

Tailwind provides margin and padding utilities to control spacing.

<div class="m-4 p-4 bg-yellow-200">This box has margin and padding</div>

Animations

Tailwind provides a set of utility classes to control animations.

<div class="animate-bounce p-4 bg-purple-400">Bouncing Box</div>

Transitions

Tailwind includes utilities to control transitions and animations.

<button class="bg-blue-500 text-white p-2 rounded-lg transition duration-300 ease-in-out hover:bg-blue-700">
  Hover me
  </button>

Shadows

Tailwind offers shadow utilities to apply shadows to elements.

<div class="shadow-lg p-4 bg-white">Large Shadow</div>

Borders

Tailwind includes a variety of border utilities to style borders.

<div class="border-4 border-dashed border-gray-500 p-4">Dashed Border</div>

Backgrounds

Tailwind provides utilities for background colors, gradients, and images.

<div class="bg-gradient-to-r from-teal-400 to-blue-500 p-4 text-white">Gradient Background</div>

Customizing Tailwind

Tailwind can be customized by modifying the tailwind.config.js file.

module.exports = {
  theme: {
    extend: {
        colors: {
        'custom-blue': '#1c3d5a',
        },
        },
        },
        plugins: [],
        }
